================================================================
  eiken_codex アプリ 起動方法
================================================================

【概要】
英検対策Web問題集アプリ。
Python の組み込みHTTPサーバーで index.html + API を提供し、
ブラウザからアクセスして模擬試験を受けられる。

----------------------------------------------------------------
■ 前提条件
----------------------------------------------------------------
- Python 3.x がインストールされていること
- プロジェクトフォルダ: D:\1_install\Git\eiken_codex

----------------------------------------------------------------
■ 初回セットアップ（questions.sqlite が存在しない場合）
----------------------------------------------------------------
CSVファイルから問題データをSQLiteデータベースにインポートする。

  cd D:\1_install\Git\eiken_codex
  python import_questions.py

  ※ デフォルトでは seed_questions.csv を読み込み、
     questions.sqlite を生成する。

  【オプション指定の場合】
  python import_questions.py --csv <CSVファイルパス> --db <DBファイルパス>

----------------------------------------------------------------
■ サーバー起動
----------------------------------------------------------------

  cd D:\1_install\Git\eiken_codex
  python server.py

  起動後、以下のメッセージが表示される:
  > Serving eiken_codex on http://127.0.0.1:8001/ with questions.sqlite

----------------------------------------------------------------
■ ブラウザでアクセス
----------------------------------------------------------------

  http://127.0.0.1:8001/

  ※ サーバーを起動したまま、上記URLをブラウザで開く。

----------------------------------------------------------------
■ サーバー停止
----------------------------------------------------------------

  コマンドプロンプト上で Ctrl + C を押す。

----------------------------------------------------------------
■ ファイル構成（主要ファイル）
----------------------------------------------------------------

  server.py           … Webサーバー本体 (ポート8001で起動)
  index.html          … アプリのフロントエンド
  styles.css          … スタイルシート
  app.js              … フロントエンド JavaScript
  import_questions.py … CSVからSQLiteへの問題インポートスクリプト
  seed_questions.csv  … 問題データ（CSVフォーマット）
  questions.sqlite    … 問題データベース（インポート後に生成）

----------------------------------------------------------------
■ APIエンドポイント（参考）
----------------------------------------------------------------

  GET  /api/test             … ランダム模擬試験を取得
  GET  /api/wrong-questions  … 過去の不正解問題一覧を取得
  POST /api/results          … 試験結果を保存

----------------------------------------------------------------
■ 試験セクション構成
----------------------------------------------------------------

  語彙      15問
  会話       5問
  整序       5問
  読解      10問
  聴解1     10問
  聴解2     10問
  聴解3     10問
  ─────────────
  合計      65問
