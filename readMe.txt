================================================================
  eikenQuest アプリ 起動方法
================================================================

【概要】
英検対策Web問題集アプリ（静的サイト版）。
サーバーなしで index.html をブラウザで開くだけで動作します。
学習記録はブラウザの LocalStorage に保存されます。

----------------------------------------------------------------
■ 使い方
----------------------------------------------------------------

1.  プロジェクトフォルダ内の `index.html` をブラウザで開きます。
    （または VS Code の Live Server などを使用してください）

2.  ブラウザで以下のURLにアクセスして学習を開始します：
    http://127.0.0.1:5500/index.html (Live Serverの場合)

----------------------------------------------------------------
■ GitHub Pages での公開
----------------------------------------------------------------

本アプリは GitHub Pages に最適化されています。
`docs/` フォルダの内容が公開用ファイルです。

公開済みURL: https://wangchang2049.github.io/eikenQuest/

----------------------------------------------------------------
■ ファイル構成
----------------------------------------------------------------

  index.html … アプリ本体
  app.js     … アプリロジック（JSONデータ読み込み & LocalStorage）
  styles.css … スタイルシート
  data/      … 問題データ（JSON形式）
  docs/      … GitHub Pages 公開用フォルダ
  qrcode.png … 公開URLのQRコード

----------------------------------------------------------------
■ 注意事項
----------------------------------------------------------------
- 学習データはブラウザごとに保存されるため、別のブラウザや端末では同期されません。
- 音声再生機能はブラウザの標準機能（Web Speech API）を使用しています。
